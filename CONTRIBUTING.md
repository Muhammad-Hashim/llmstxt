# Contributing

Thanks for taking the time to contribute.

## Development

Requirements:
- Node.js 18+
- npm

Install:
```bash
npm install
```

Test:
```bash
npm test
```

Build:
```bash
npm run build
```

## Pull Requests

- Keep changes focused and well-scoped.
- Add/adjust tests when behavior changes.
- Ensure `npm test` and `npm run build` pass.
